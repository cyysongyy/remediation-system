# 🎯 Remedial System v3.0

拍照上傳考卷 → AI 自動批改 → 找出迷思概念 → 生成補救題目

**線上使用：** https://cyysongyy.github.io/remedial-system/

## 🚀 快速開始

### 第一步：取得免費 API Key

前往 [Google AI Studio](https://aistudio.google.com/apikey) → 點擊 **Create API key**

- ✅ 完全免費，每日最高 **1,440 次**請求
- ✅ 無需綁定信用卡
- Key 格式：`AQ.Ab8R…` 或 `AIzaSy…` 兩種都支援

### 第二步：開啟系統

點擊上方線上連結，或下載 `index.html` 用瀏覽器開啟。

### 第三步：設定 API Key

進入 **⚙️ 設定** → 貼上 Gemini API Key → 點擊**儲存**

### 第四步：開始批改

1. 點選 **📷 批改考卷** → 建立考卷（填入科目、年級、學年度）
2. 上傳學生考卷照片
3. 若一位學生有多頁：上傳完第一位學生的所有頁面後，點擊 **＋ 新學生分隔**，再上傳下一位學生的照片
4. 點擊 **🤖 開始 AI 批改**，系統逐位學生分析

## ☁️ 雲端同步設定（選填）

若需將批改記錄同步至 Google 試算表，請依以下步驟設定：

1. 開啟一份 Google 試算表 → **擴充功能** → **Apps Script**
2. 將 `remediation-backend.gs` 的內容全部貼上（取代原有內容）
3. 點擊**部署** → 新增部署作業 → 網頁應用程式
   - 以下列身分執行：**我（Me）**
   - 誰可以存取：**所有人（Anyone）**
4. 複製部署網址
5. 回到系統：**⚙️ 設定** → 貼上網址 → **🔌 測試連線**

後台會自動建立兩個工作表：
- **批改記錄** — 所有學生批改結果
- **考卷設定** — 考卷基本資料

## 🤖 AI 引擎說明

| 提供商 | 模型 | Key 格式 | 費用 |
|---|---|---|---|
| Google Gemini | `gemini-2.5-flash`（預設） | `AQ.…` 或 `AIzaSy…` | 免費（每日1440次） |
| OpenAI | `gpt-4o-mini` | `sk-…` | 付費 |

兩組 Key 可同時設定，在 **⚙️ 設定** 中選擇優先使用哪個引擎（Gemini / OpenAI / 自動）。

多頁考卷會將所有頁面**一次送入 AI**，讓 AI 綜合所有頁面後再評分，提升準確度。

## 💾 資料儲存說明

- **本機模式**：所有資料預設存於瀏覽器 `localStorage`，清除瀏覽器資料會遺失
- **雲端模式**：設定 GAS 後台後可同步至 Google 試算表，跨裝置查看
- **隱私保護**：API Key 僅儲存於您自己的瀏覽器，直接呼叫 Gemini / OpenAI，不經過任何第三方伺服器
