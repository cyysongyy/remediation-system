🎯 Remedial System v3.0
拍照上傳考卷 → AI 自動批改 → 找出迷思概念 → 生成補救題目
🚀 使用流程
1
⚙️ 設定 AI API Key
進入「設定」頁面，輸入 Gemini 或 OpenAI 的 API Key。
建議使用 Google Gemini 2.5 Flash（免費，每日1440次：aistudio.google.com）
2
📝 建立考卷
「批改考卷」→ 填入學年度、學期、名稱、科目、年級。
選填 上傳答案卷照片（老師版），讓 AI 自動比對正確答案。
選填 輸入班級名單（座號 + 姓名），AI 可自動辨識對應學生。
3
📷 上傳學生考卷照片
選取考卷後，可以：
• 一次選多張照片（多位學生同時批改）
• 拖放照片到上傳區
• 用相機即時拍攝
點「🤖 開始 AI 批改」，AI 逐張分析並即時顯示結果。
4
📊 查看分析結果
每位學生批改完成後，系統顯示：
• 分數圓環、答錯題號
• 迷思概念標籤
• 補救建議
進入「班級分析」可查看整班統計圖表。
5
💡 生成補救題目
「補救題庫」頁面會自動列出所有批改記錄中出現的迷思概念，
點「🤖 生成題目」即自動生成對應練習題（含解析）。
也可手動指定科目與迷思概念生成。

❓ 常見問題
AI 沒有答案卷怎麼批改？▾
AI 會根據題目內容與常見教學知識推斷正確答案。建議上傳答案卷照片可大幅提升準確度。
一次可以上傳幾張照片？▾
理論上不限張數，但建議每批不超過 20 張，避免等待時間過長。
資料存在哪裡？▾
批改記錄預設存在本機瀏覽器（localStorage）。設定 GAS 後台後可同步至 Google 試算表。
API Key 安全嗎？▾
API Key 只存在你自己的瀏覽器，不會傳送給任何第三方，只用於直接呼叫 Google / OpenAI API。
手機可以用嗎？▾
可以，支援手機瀏覽器，相機功能會調用後置鏡頭直接拍照。
